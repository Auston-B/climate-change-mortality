# WHO Mortality — raw data (not committed)

The six raw WHO mortality files are excluded from version control because of their size
(~442 MB total):

```
Morticd10_part1
Morticd10_part2
Morticd10_part3
Morticd10_part4
Morticd10_part5
Morticd10_part6
```

Download them from the **WHO Mortality Database** and place them in this folder to run
the notebook end-to-end:

https://www.who.int/data/data-collection-tools/who-mortality-database

`country_codes` (the code → country-name lookup) **is** committed here, so it is
available out of the box.
